# Parser app

## Поддерживаемые и потенциальные форматы

Единый источник истины для форматов находится в `parser_app/parsers/formats.py`.

### SUPPORTED_FORMATS

Фактически поддерживаемые форматы:

- `pdf`
- `docx`
- `doc`
- `txt`
- `csv`
- `json`
- `md`
- `markdown`
- `tsv`
- `djvu`
- `xls`
- `xlsx`
- `odt`
- `ods`

### POTENTIALLY_USEFUL_FORMATS

Форматы-кандидаты для будущей поддержки:

- `rtf`
- `epub`
- `html`
- `xml`
- `yaml`
- `ppt`
- `pptx`
- `odg`
- `pages`
- `numbers`
- `eml`
- `msg`
